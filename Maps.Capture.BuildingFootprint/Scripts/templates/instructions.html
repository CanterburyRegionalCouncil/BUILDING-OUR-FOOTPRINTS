To start mapping:

  1. Click the Start editing button
  2. Click on one of the building type templates above e.g. Residential Building
  3. Click on the map at a point on the outline of a building to start using the tool
  4. Click at each corner to create a vertex, or angle
  5. Double click to finish shape. Note: You do not have to finish on the start point - it will automatically join up the shape for you
  6. Fill out other details about the building on the popup

Trust rating not high enough? Try to zoom in further when digitising.
